# 🥁 Drum Kit Website

A fun and interactive **Drum Kit web application** built using **HTML, CSS, and JavaScript**.  
Users can play different drum sounds either by **clicking buttons** or by **pressing keyboard keys**.

This project is part of **The Complete Web Development Bootcamp (Udemy)** and helped me understand DOM manipulation, event handling, and JavaScript logic.

---

## 🚀 Live Demo
👉 (https://prarthanabhandari.github.io/Drump-Kit/)

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- DOM Manipulation
- Event Listeners
- Audio API

---

## 🎮 Features
- 🎵 Play drum sounds on **mouse click**
- ⌨️ Play drum sounds using **keyboard keys**
- 🔊 Real drum sound effects
- ⚡ Fast and responsive interaction
- 📱 Simple and clean UI

---

## 🧠 What I Learned
- How to use `addEventListener()` for click and keyboard events
- Handling keyboard input using `keydown`
- Using `this` keyword in JavaScript
- JavaScript `switch` statements
- Working with audio files using `new Audio().play()`
- DOM selection using `querySelectorAll()`
